# monitor-deportivo-dash
¿Qué hace esta app?
Solución integral de monitorización deportiva diseñada para corredores de todos los niveles. Visualiza en tiempo real frecuencia cardíaca (ECG), VO₂ max, calorías consumidas y nivel de fatiga. Incluye panel para profesionales de la salud con gestión de pacientes y análisis de rendimiento.

Tecnologías usadas

Python + Dash + Plotly

JSON para persistencia de datos

Procesamiento de señales fisiológicas
